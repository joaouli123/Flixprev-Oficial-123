# Produto

## FlixPrev I.A

Plataforma de Inteligência Artificial para Advocacia Previdenciária.

## Características Principais

- Sistema de gerenciamento de agentes de IA
- Autenticação e autorização de usuários via Supabase
- Painel administrativo para gestão de usuários e conteúdo
- Sistema de assinaturas com controle de status (ativo/desativado)
- Progressive Web App (PWA) com suporte offline
- Rastreamento de eventos via Facebook Pixel e CAPI
- Suporte a temas claro/escuro
- Interface responsiva e acessível

## Público-Alvo

Advogados e profissionais da área previdenciária que utilizam IA para otimizar seu trabalho.

## Idioma

Português brasileiro (pt-BR) - toda a interface e conteúdo são em português.
